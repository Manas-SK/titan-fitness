# Titan Fitness 💪

A modern gym and fitness website built using React and Vite.

## Features

* Responsive Design
* Gym Membership Plans
* Trainer Profiles
* BMI Calculator
* Testimonials Section
* Contact Form
* Modern Black & Gold UI

## Technologies Used

* React
* Vite
* JavaScript
* CSS3
* Git & GitHub

## Installation

```bash
npm install
npm run dev
```
